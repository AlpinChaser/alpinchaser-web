# Feature-Anforderungen – AlpinChaser / PassAtlas

Diese Datei dokumentiert geplante Features und Anforderungen für das Projekt.

---

## User Bewertungen je Pass – Live-Verkehrsinfos

**Erstellt:** 2026-04-11

User Bewertungen pro Pass sollen erweitert werden, damit Nutzer Meldungen zu Verkehr, Straßenzustand usw. abgeben können.

**Ziel:** Live-Informationen anzeigen, z.B. „vor 5 Minuten viel Verkehr gemeldet" — ähnlich wie Community-basierte Verkehrsmeldungen (Waze-Style).

**Anforderungen:**
- Bewertungskategorien erweitern: Verkehr, Straßenqualität, ggf. Wetterbedingungen
- Zeitstempel je Meldung speichern
- Anzeige der jüngsten Meldungen pro Pass (z.B. letzte 60 Minuten)
